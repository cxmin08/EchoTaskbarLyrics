# Echo Taskbar Lyrics

Echo Taskbar Lyrics 是一个面向 Windows 任务栏的歌词显示工具，可配合 EchoMusic 使用，
将当前播放歌词嵌入任务栏区域显示。

## 主要功能

- 支持卡拉 OK 模式，显示单行歌词、逐字高亮、翻译和长歌词跑马灯。
- 支持卡片模式，显示专辑封面、当前歌词和第二行内容。
- 支持卡片模式封面左侧或右侧布局。
- 支持水平任务栏。
- 支持全屏自动隐藏、拖动位置、开机自启和托盘设置入口。

## 兼容性

项目主要面向 Windows 10 任务栏体验设计，优先保证 Windows 10 下的稳定显示、
定位和任务栏空间适配。

- Windows 10：主要支持目标，推荐使用。
- Windows 11：次要支持目标，基础显示、设置和托盘功能可用。
- 仅支持水平任务栏。
- Windows 11 的开始菜单、搜索面板和任务栏动画可能导致任务栏区域短暂变化，程序会尽量冻结稳定位置以减少跳动。

## 项目来源

本项目基于 [Yngu196/TaskbarLyrics](https://github.com/Yngu196/TaskbarLyrics)
二次开发。

原项目是 MeoKoeMusic 的任务栏歌词插件；本项目在此基础上改造为
EchoMusic 的任务栏歌词插件，并调整了连接方式、设置项和显示模式。

## 显示模式

### 卡拉 OK 模式

默认显示模式，适合只关注当前歌词的场景。

- 可开启或关闭逐字高亮。
- 可显示当前行翻译。
- 长歌词可使用跑马灯滚动，支持往返滚动、循环滚动和关闭滚动。

### 卡片模式

卡片模式在任务栏中显示封面和双行歌词，适合同时查看歌曲信息和歌词上下文。

- 第一行固定显示当前歌词。
- 第二行默认显示下一句歌词。
- 在卡片模式设置中开启“显示翻译”后，第二行优先显示当前行翻译；没有翻译时回退显示下一句歌词。
- 当前行歌词过长时会复用“长歌词滚动（跑马灯）”配置，按卡片实际文字区域宽度滚动，避免歌词被封面或窗口边界遮挡。
- 封面可放在左侧或右侧，跑马灯会按对应文字区域计算可用宽度。

## 通信说明

- EchoMusic 插件通过固定的本地 WebSocket 端口 `127.0.0.1:6523` 与原生辅助程序通信；该端口仅监听回环地址并使用随机 Token 鉴权，属于内部实现，不再作为用户设置暴露。
- `6520` 仅用于直接运行 EXE 时连接 EchoMusic 旧 WebSocket API；由 EchoMusic 插件启动时不会使用该端口，也不再显示 WebSocket 端口设置。
- 歌词内容按 `ctx.nowPlaying.onSnapshot` 事件通过持久 WebSocket 同步，原生端在本地推算播放进度；插件每秒发送一次歌词心跳以便原生窗口在被 Shell 遮挡后及时恢复置顶，播放控制命令通过同一 WebSocket 连接实时下发。

## 设置说明

打开托盘菜单中的设置后，可以调整：

- 显示模式和窗口宽度。
- 卡拉 OK 模式的字体、颜色、透明度、翻译和跑马灯。
- 卡片模式的当前行字号、第二行字号、字体、颜色、显示翻译和封面位置。
- 位置偏移、刷新率、调试日志和开机自动启动。

## 构建

项目使用 CMake 和 vcpkg 构建。可使用仓库根目录的构建脚本：

```bat
build.cmd release
```

或使用 CMake 预设：

```bat
cmake --build --preset x64-Release
```
